About this Game
This is a simple, mobile-friendly, farming RPG game where you start a farm, fish, craft and explore the world of the game.

This game is free-to-play and doesn't require any purchases to progress and enjoy the content. However, some features come out in "Beta" for 7-14 days and are then released to all players. "Beta" features are available to those that support the game via Patreon.
